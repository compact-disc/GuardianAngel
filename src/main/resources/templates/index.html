Welcome to Guardian Angel - Alpha!

Guardian Angel is an application that collects Twitter data to analyze. It uses The Stanford Natural Language Processing Group’s CoreNLP library to scan tweets for sentiment analysis. Each tweet is given a score ranging from zero to four. Zero is the most negative and four being the most positive score given. Each tweet is also given a sentiment type ranging from very negative to very positive.

By clicking the "Pull Tweets" button you are consenting to have all of your recent Twitter data pulled directly from Twitter and stored in the Guardian Angel database. After clicking the “Pull Tweets” button all of the tweets collected and analyzed will be displayed on the "View Tweets" page.
